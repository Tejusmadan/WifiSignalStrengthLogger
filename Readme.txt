# Wi‑Fi Scanner App (Jetpack Compose)

A **Compose-based Android app** for scanning nearby Wi‑Fi Access Points (APs) across multiple locations.

---

## ✨ Features

- Scan nearby Wi‑Fi APs **100× per user-selected location** (3 tabs)
- Displays each AP’s **RSSI range** (min..max)
- **Expandable 10×10 sample matrix** per AP
- **Comparison tab** shows side-by-side RSSI ranges across locations
- **Real-time linear progress bar** during scans

---

## 🧰 Prerequisites

- Android Studio
- Android device or emulator
- Min SDK: **24**
- Target SDK: **35**

### Required Permissions

- `ACCESS_FINE_LOCATION`
- `ACCESS_WIFI_STATE`
- `CHANGE_WIFI_STATE`
